Sobre Mí

Soy un desarrollador frontend con enfoque en la creación de interfaces de usuario usables e interactivas para sitios y aplicaciones web complejas. Combino diseño centrado en el usuario con desarrollo eficiente, encargándome de la maquetación y el desarrollo de interfaces modernas que priorizan el rendimiento y la experiencia del usuario.

Utilizo Figma para el diseño y prototipado de interfaces, y trabajo principalmente con React y TypeScript para construir productos web robustos, escalables y mantenibles.
